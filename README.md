# customer_behavior_analysis
Data analysis project showcasing customer behavior analysis using python , sql and power Bi.
Project Steps
1. Data Loading
Imported dataset using Python libraries.
Verified structure, columns, and data types.
2. Exploratory Data Analysis (EDA)
Analyzed distributions and relationships.
Identified trends, patterns, and anomalies.
Visualized data using charts and graphs.
3. Data Cleaning
Handled missing values.
Removed duplicates.
Fixed inconsistent data formats.
Prepared data for analysis and SQL operations.
4. SQL Analysis (PostgreSQL)
Imported cleaned data into PostgreSQL.
Wrote SQL queries to:
Aggregate data
Filter records
Perform joins and calculations
Extracted insights from structured queries.
5. Dashboard Creation (Power BI)
Built an interactive dashboard.
Included key metrics, charts, and filters.
Designed for easy understanding and usability.
6. Reporting
Compiled insights into a structured report.
Highlighted key findings and business implications.
7. Presentation (Gamma)
Created a professional presentation.
Summarized project workflow and insights.
Designed for clear communication to stakeholders.
